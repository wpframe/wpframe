As I'm a university student, most of my repositories are privated due to academic integrity guidelines. Here's a rundown on them:
- cosc326: My work done in the Computation Problem Solving (COSC326) paper at the University of Otago. This repository features code for 12 assignments in which we used Python, Java, C, and C++ to solve programming problems.
- cosc343: My repository for the Artifical Intelligence (COSC343) paper at the University of Otago. It holds code and reports on two assignments; in one we were required to create an agent to solve [Mastermind](https://en.wikipedia.org/wiki/Mastermind_(board_game)) puzzles, and in one we were required to create a genetic algorithm to train cleaners in a vacuum cleaner game.
- ANDIE: A group project from the Software Development (COSC202) paper at the University of Otago. ANDIE is A Non Destructive Image Editor made in Java. We were required to implement additional features to this application. I worked on the rotate, flip, and resize image features, keyboard shortcuts, macros, and automated image uploading.
- 6809assembly: Simply holds the assembly code I wrote for an assignment in a Computer Systems paper (COSC204) for the Motorola 6809 chip.

Also privated is the repository for rymBot, a Discord bot I wrote to quickly serve users links to artists, genres, and music releases on the RateYourMusic website. It also provides pretty embeds for the current song users are listening to on request. I intended for this bot to make use of the RateYourMusic API right after release, however there have been no updates on the status of the API's release for years. rymBot serves Discord servers totalling over 7500 users and stores data for over 1000 users.
